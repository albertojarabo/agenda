# Agenda

Proyecto demo para crear parte backend de servicio de aplicacion agenda.

### Partes abordadas en el proyecto:

- Modelo
- Persistencia en BBDD
- Servicio REST para CRUD


### Librerias:

- Spring Boot
- Spring MVC
- Spring Data
- JPA
- Lombok


###Incluye:

- Paquete con pruebas para ejecutar desde Postman
